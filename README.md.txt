# 🌳 PRODIGY_DS_TASK03 - Decision Tree Classifier

## 📌 Project Overview

This project is part of the **Prodigy InfoTech Data Science Internship**.

The objective of this task is to build a **Decision Tree Classifier** to predict whether a customer will purchase a product or service based on demographic and behavioral data using the **Bank Marketing Dataset**.

---

## 🎯 Objective

Develop a machine learning model using the Decision Tree algorithm to classify customer purchase decisions based on their demographic and marketing campaign information.

---

## 📂 Dataset

* **Dataset:** Bank Marketing Dataset
* **Source:** UCI Machine Learning Repository
* **Target Variable:** `y` (Customer Purchase)

The dataset contains customer details such as age, job, marital status, education, balance, loan information, contact type, campaign details, and previous marketing outcomes.

## 📋 Project Workflow
